Vector Map

This map was meant to show flower density throughout the city of Utrecht, with information collected by the class. Unfortunately, this map did not end up working. Several issues arose while producing this map. It was created using ArcGIS Online through ESRI. I completed a course regarding the use of the application and received a certificate, but the data I was trying to work with was incompatible.

The first issue was unable to be fixed on my end, as the data included flower type, and I was unable to switch it to a qualitative view of flower density. The second issue was the lack of overall data. This was the biggest drawback because when I tried to calculate flower hotspots, every result was not significant (see map). Thirdly, I did not have enough credit to run any analysis with additional data, so I could not compare the flower hot spot to population density, for example.
